- **프로젝트 개요:** 순수 HTML5와 CSS3만으로 구현한 모바일 최적화 길 안내 웹 UI입니다.
- **핵심 구조:** 화면 스크롤을 방지하고 레이어를 겹치는(Layered) 방식으로 네이티브 앱과 같은 사용자 경험을 제공합니다.
- **UI/UX 디자인:** 시인성 높은 폰트와 직관적인 대형 마이크 버튼으로 조작 편의성을 극대화했습니다.
- **레이아웃 기술:** CSS Flexbox를 활용해 기기 화면에 맞춰 모든 요소를 깔끔하게 수직/수평 정렬했습니다.
- **빠른 실행:** 별도의 서버 세팅이나 라이브러리 설치 없이 `first.html` 파일을 열어 즉시 구동할 수 있습니다.
